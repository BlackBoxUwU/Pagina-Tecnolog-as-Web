Tecnologías de Web APIs

Definición de API

Las APIs permiten la comunicación entre diferentes sistemas de software a través de protocolos bien definidos.
Actúa como una capa intermedia que procesa las transferencias de datos entre sistemas, permitiendo a las empresas abrir sus datos y funcionalidades de aplicaciones a desarrolladores externos, socios comerciales y departamentos internos dentro de sus empresas

Tipos de APIs Web

    Existen diferentes tipos de APIs, cada una pensada para distintos contextos de uso.
  • REST: Es la más utilizada por su simplicidad y flexibilidad. Está basada en operaciones HTTP como GET, POST, PUT y DELETE.
    Funciona con formatos como JSON y XML, aunque JSON es el más común por su ligereza y facilidad de uso.
    Es ideal para aplicaciones web y móviles que consumen datos de manera sencilla.
  • SOAP: Aunque más complejo, es muy robusto para transacciones críticas, por ejemplo en sistemas financieros o de salud.
    Usa exclusivamente XML para el intercambio de datos y requiere estructuras bien definidas (WSDL).
    Ofrece estándares de seguridad integrados como WS-Security.

  • JWT (JSON Web Token): No es un tipo de API en sí, sino una técnica muy usada para autenticar y autorizar accesos a APIs web.
    Consiste en un token codificado en base64 que contiene información del usuario y tiene firma digital para evitar manipulaciones.
    Es ampliamente usado en APIs REST para manejar sesiones sin necesidad de almacenar estado en el servidor.
  • GraphQL: Desarrollado por Facebook, permite a los clientes consultar solo los datos que necesitan, lo que optimiza la eficiencia y reduce la sobrecarga.
    Opera con un único endpoint y permite validación automática del esquema.
    Muy usado en aplicaciones modernas con interfaces dinámicas como React o Angular.
  • gRPC: Creado por Google, es ideal para la comunicación eficiente entre microservicios.
    Utiliza HTTP/2 y Protobuf (Protocol Buffers) para serializar los datos, lo que lo hace más rápido que JSON.
    Soporta llamadas en tiempo real, streaming y múltiples lenguajes de programación.
